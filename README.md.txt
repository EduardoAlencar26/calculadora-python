Calculadora em Python

Autor: Eduardo Alencar da Silva

Sobre o Projeto

Calculadora desenvolvida em Python que realiza:

- Adição
- Subtração
- Multiplicação
- Divisão (com tratamento de erro)

---

Como Executar no Windows

1. Instale o Python
2. Abra o terminal na pasta do projeto
3. Execute:

python calculadora.py

---

Como Executar no Linux/Mac

1. Abra o terminal na pasta
2. Execute:

chmod +x calculadora.sh
./calculadora.sh

---

Explicação do Código

O programa contém:

- Função `calcular()` → Executa a operação escolhida.
- Estrutura `if/elif` → Define qual cálculo será feito.
- `try/except` → Evita erro se o usuário digitar algo inválido.
- `while True` → Mantém o programa rodando até o usuário sair.
